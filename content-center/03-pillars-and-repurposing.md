# PART 2 — WEEKLY CONTENT REPURPOSING SYSTEM

## The Machine (repeatable process)

One **core idea** per week becomes ~8 assets across LinkedIn (personal + business page + carousel), video, Instagram, TikTok, email, and outreach. The core idea is always product-forward and maps to a pillar.

**The 6-step weekly loop:**

1. **Pick the core idea (Mon).** One sentence, one pillar, one buyer takeaway. Must reference a real product surface (Snapshot, Opportunity Web, Dashboard, a node, or the pilot offer).
2. **Write the anchor asset first (Mon).** The LinkedIn personal post is the anchor — richest reasoning lives here. Everything else is a re-cut, not a rewrite.
3. **Split by format logic (Tue):**
   - *Business page post* = same idea, org voice, product-forward, ends on the offer.
   - *Carousel* = the idea broken into a slide sequence (hook → tension → frame → product → CTA).
   - *Video/TikTok* = the idea as a spoken hook + 2–3 beats + CTA.
   - *Instagram* = the emotional/visual cut + save-worthy caption.
4. **Cascade CTAs by platform (Tue).** LinkedIn/email → signup link. IG/TikTok → "link in bio." DM → soft, conversational, no link until they engage.
5. **Schedule (Wed):** LinkedIn personal + page early week; carousel mid-week; video Thu; IG/TikTok Fri; email + DMs run continuously against warm engagers.
6. **Harvest engagement (Fri):** every commenter/DM-er becomes an outreach thread. Comments and replies feed next week's core idea.

**Golden rules:** never invent a Snapshot outcome as a guarantee · never call the Snapshot free · always keep teal=strength / gold=gap / rose=deadline color logic accurate · one link, everywhere: https://anansiatlas.com/anansi-atlas/
